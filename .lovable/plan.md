

# План: Убрать рамку у блоков treemap

## Текущее состояние

Каждый блок treemap имеет белую полупрозрачную рамку в 1 пиксель:

```typescript
border: '1px solid rgba(255,255,255,0.3)',
```

## Изменение

Убрать строку `border` из стилей `motion.div` в `TreemapNode.tsx`.

## Файл для изменения

| Файл | Изменение |
|------|-----------|
| `src/components/treemap/TreemapNode.tsx` | Удалить строку 169: `border: '1px solid rgba(255,255,255,0.3)'` |

## Результат

Блоки будут отображаться без рамок — только с фоновым цветом и скруглёнными углами (borderRadius: 4).

## Оценка

| Метрика | Значение |
|---------|----------|
| Сложность | Минимальная |
| Строк изменится | 1 |
| Риск регрессии | Нулевой |
